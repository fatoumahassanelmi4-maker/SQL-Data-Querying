# 📊 SQL Data Analysis Projects (PostgreSQL)

## 📌 Project Overview

This repository contains SQL analysis projects completed using **PostgreSQL and pgAdmin**.  
The objective was to practice SQL querying, data analysis, and business problem-solving using two different datasets:

- **Chinook Database** – Music store relational database
- **Sales Data** – Sales transaction dataset

The projects demonstrate how SQL can be used to extract insights, analyze trends, and support data-driven decisions.
